| Состояние | Визуализация |

```
Login → [Success] → Dashboard
Login → [Forgot Password] → Forgot Password
```

## 8. Responsive стратегия

### 8.1 Breakpoints

| Breakpoint | Ширина | Устройства | Sidebar |
|------------|--------|------------|---------|
| Desktop XL | ≥ 1920px | Large monitors | Expanded |
| Desktop | 1280px - 1919px | Standard monitors | Expanded |
| Laptop | 1024px - 1279px | Laptops | Collapsible |
| Tablet | 768px - 1023px | Tablets | Hidden, hamburger menu |
| Mobile | < 768px | Mobile phones | Hidden, hamburger menu |

### 8.3 Responsive компоненты

| Компонент | Desktop | Tablet | Mobile |
|-----------|---------|--------|--------|
| Table | Горизонтальный скролл | Горизонтальный скролл | Вертикальный + expandable rows |

| Элемент | Desktop | Mobile |
