# Changelog

## 0.1.0 - MVP release

### Added

- Authentication and RBAC.
- Admin module for users, roles, permissions, settings, and audit logs.
- KPI and reporting screens.
- Documents and attachments subsystem.
- Observability: JSON logs, metrics, health checks, optional error tracking.
- Production-ready docs and release runbooks.

### Notes

- Swagger remains disabled by default in production.
- HTTP audit logging and error tracking are opt-in.
- Release requires database migration review and backup.
